# 定时器-间歇函数

## 介绍

网页中经常会需要一种功能：每隔一段时间需要自动执行一段代码，不需要我们手动去触发

例如：网页中的倒计时

要实现这种需求，需要定时器函数

定时器函数有两种，分别是：

- **间歇函数 setInterval**：每隔指定时间，重复执行一段代码
- **延时函数 setTimeout**：等待指定时间后，只执行一次代码

## 基本使用

### 开启定时器

```
setInterval(函数，间隔时间 )
```

- 每隔一段时间调用这个函数
- 间隔时间是毫秒

例：

1.

```
<body>
  <script>
    // setInterval(函数, 间隔时间)
    setInterval(function () {
      console.log('一秒执行一次')
    }, 1000)
  </script>
</body>
```

2.

```html
<body>
  <script>
    function fn() {
      console.log('一秒执行一次')
    }
    setInterval(fn, 1000)
  </script>
</body>
```

注意：

1. 函数名字不需要加括号
2. 定时器返回的是一个id数字

### 关闭定时器

```
let 变量名 = setInterval(函数, 间隔时间)
clearInterval(变量名)
```

一般不会刚创建就停止，而是满足一定条件再停止

```javascript
let timer = setInterval(function() {
  console.log('hi~~')
}, 1000)
clearInterval(timer)
```



# 需求
实现阅读协议倒计时按钮效果
1. 页面有文本域与禁用状态的倒计时按钮
2. 按钮默认不可点击，显示剩余倒计时秒数
3. 每秒秒数递减，同步更新按钮文字
4. 倒计时结束，清除定时器，按钮恢复可点击状态，修改按钮文字

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <title>阅读协议倒计时按钮</title>
</head>
<body>
  <textarea></textarea>
  <br>
  <button class="btn" disabled>我已经阅读用户协议(5)</button>

  <script>
    // 1. 获取按钮元素
    const btn = document.querySelector('.btn')

    // 2. 定义倒计时初始秒数
    let i = 5

    // 3. 开启定时器
    let n = setInterval(function () {
      i--
      // 更新按钮文字
      btn.innerHTML = `我已经阅读用户协议(${i})`

      // 倒计时结束处理
      if (i === 0) {
        clearInterval(n) // 关闭定时器
        btn.disabled = false // 启用按钮
        btn.innerHTML = '我已阅读并同意用户协议' // 更新最终文字
      }
    }, 1000)
  </script>
</body>
</html>
```
**表单输入框（input/textarea）用 value，不能用 innerHTML**

**button 文字用 innerHTML**
