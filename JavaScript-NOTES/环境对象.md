# 环境对象

环境对象：指的是函数内部特殊的`变量this`,代表当前函数运行时所处的环境

核心原则

**谁调用，this 就是谁**

 1.普通函数中的 this

直接调用函数，等价于 `window.函数名()`
所以普通函数内的 this 指向 **window**

> 严格模式下普通函数 this 为 `undefined`

 2.对象方法调用

格式：`对象.方法()`
this 指向 **调用方法的对象**

3. new 构造函数调用

this 指向 **new 创建的实例对象**

4. 箭头函数

**没有自己的 this**，继承外层作用域的 this
