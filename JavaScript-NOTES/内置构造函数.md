# 内置构造函数

在 JavaScript 中最主要的数据类型有 6 种：

基本数据类型：

- 字符串、数值、布尔、undefined、null

引用类型：

- 对象

但是，我们会发现有些特殊情况：

```
// 普通字符串
const str = 'andy'
console.log(str.length) // 4
```

其实字符串、数值、布尔等基本类型也都有专门的构造函数，这些我们称为包装类型。
JS 中几乎所有的数据都可以基于构造函数创建。

## Object

Object 是内置的构造函数，用于创建普通对象。

```js
// 通过构造函数创建普通对象
const user = new Object({name: '小明', age: 15})
```

1.Object.keys

- 作用：Object.keys 静态方法获取对象中所有属性（键）
- 语法：

```js
const o = { name: '佩奇', age: 6 }
// 获得对象的所有键，并且返回是一个数组
const arr = Object.keys(o)
console.log(arr) // ['name', 'age']
```

- 返回的是一个数组

2.Object.values

- 作用：Object.values 静态方法获取对象中所有属性值
- 语法：

```javascript
const o = { name: '佩奇', age: 6 }
// 获得对象的所有值，并且返回是一个数组
const arr = Object.values(o)
console.log(arr) // ['佩奇', 6]
```

3.Object.assign

- 作用：Object.assign 静态方法常用于对象拷贝
- 语法：

```javascript
// 拷贝对象 把 o 拷贝给 obj
const o = { name: '佩奇', age: 6 }
const obj = {}
Object.assign(obj, o)
console.log(obj) // {name: '佩奇', age: 6}
```

### 5. `join` 语法与示例

**作用**：将数组元素拼接为字符串，返回拼接后的字符串（默认用逗号分隔）。

```javascript
// 语法
数组.join(分隔符)

// 示例
const arr = ['佩奇', '乔治', '猪爸爸']
console.log(arr.join())      // "佩奇,乔治,猪爸爸"（默认逗号）
console.log(arr.join(' - ')) // "佩奇 - 乔治 - 猪爸爸"（自定义分隔符）
console.log(arr.join(''))    // "佩奇乔治猪爸爸"（无分隔符）
```

------

### 6. `find` 语法与示例

**作用**：查找符合条件的第一个数组元素，返回该元素值；无匹配时返回 `undefined`。

```javascript
// 语法
数组.find(function(当前元素, 索引, 原数组) {
  // 返回 true 表示找到目标元素
})

// 示例
const arr = [
  { name: '佩奇', age: 6 },
  { name: '乔治', age: 4 },
  { name: '佩奇', age: 10 }
]
// 找到第一个 age 为 6 的元素
const target = arr.find(item => item.age === 6)
console.log(target) // { name: '佩奇', age: 6 }

// 无匹配时返回 undefined
const noTarget = arr.find(item => item.age === 100)
console.log(noTarget) // undefined
```

------

### 7. `every` 语法与示例

**作用**：检测数组所有元素是否都符合条件，全部通过返回 `true`，否则返回 `false`。

```javascript
// 语法
数组.every(function(当前元素, 索引, 原数组) {
  // 返回 true/false 表示当前元素是否符合条件
})

// 示例
const arr1 = [18, 20, 22]
const arr2 = [16, 20, 22]

// 所有元素都 >= 18？
const allAdult1 = arr1.every(num => num >= 18)
console.log(allAdult1) // true

const allAdult2 = arr2.every(num => num >= 18)
console.log(allAdult2) // false（因为16不满足）
```

## String

在 JavaScript 中的字符串、数值、布尔具有对象的使用特征，如具有属性和方法

```javascript
// 字符串类型
const str = 'hello world!'
// 统计字符的长度（字符数量）
console.log(str.length)

// 数值类型
const price = 12.345
// 保留两位小数
price.toFixed(2)
```

之所以具有对象特征的原因是字符串、数值、布尔类型数据是 JavaScript 底层使用 Object 构造函数 “包装” 来的，被称为**包装类型**。

### 1. 常见实例方法

1. 实例属性 `length`：用来获取字符串的长度(重点)
2. 实例方法 `split('分隔符')`：用来将字符串拆分成数组(重点)
3. 实例方法 `substring(需要截取的第一个字符的索引[, 结束的索引号])`：用于字符串截取(重点)
4. 实例方法 `startsWith(检测字符串[, 检测位置索引号])`：检测是否以某字符开头(重点)
5. 实例方法 `includes(搜索的字符串[, 检测位置索引号])`：判断一个字符串是否包含在另一个字符串中，根据情况返回 `true` 或 `false`(重点)
6. 实例方法 `toUpperCase`：用于将字母转换成大写
7. 实例方法 `toLowerCase`：用于将字母转换成小写
8. 实例方法 `indexOf`：检测是否包含某字符
9. 实例方法 `endsWith`：检测是否以某字符结尾
10. 实例方法 `replace`：用于替换字符串，支持正则匹配
11. 实例方法 `match`：用于查找字符串，支持正则匹配

下面是前四个字符串属性/方法的语法和示例：

------

### 1. `length`（实例属性）

**作用**：获取字符串的长度（字符个数）

```javascript
const str = 'hello'
console.log(str.length) // 输出：5
```

------

### 2. `split('分隔符')`（实例方法）

**作用**：按指定分隔符把字符串拆成数组

```javascript
const str = '苹果,香蕉,橘子'
const arr = str.split(',')
console.log(arr) // 输出：['苹果', '香蕉', '橘子']

// 不传参数：返回整个字符串作为数组唯一元素
console.log(str.split()) // ['苹果,香蕉,橘子']

// 空字符串''做分隔符：按每个字符拆分
console.log('abc'.split('')) // ['a','b','c']
```

------

### 3. `substring(开始索引[, 结束索引])`（实例方法）

**作用**：截取字符串，包含`开始索引`字符，不包含`结束索引`字符；省略结束索引则截取到末尾

```javascript
const str = 'hello world'

// 从索引 0 截取到索引 5（不包含5）
console.log(str.substring(0, 5)) // 'hello'

// 从索引 6 截取到末尾
console.log(str.substring(6)) // 'world'
```

------

### 4. `startsWith(检测字符串[, 检测位置])`（实例方法）

**作用**：检测字符串是否以指定内容开头，返回布尔值

```javascript
const str = 'https://www.example.com'

// 是否以 'https' 开头
console.log(str.startsWith('https')) // true

// 从索引 6 开始检测是否以 'www' 开头
console.log(str.startsWith('www', 6)) // true
```

## Number

Number 是数值的内置构造函数，属于包装类型，普通数值会被底层自动包装为 Number 对象，从而可以调用属性和方法。

### 一、通过构造函数创建数值
```javascript
// 字面量创建（常用）
const num1 = 10

// 构造函数创建数值
const num2 = new Number(20)
console.log(num2) // [Number: 20]
```

### 二、常见实例属性
#### 1. `length`
Number 实例无实际有效 length 属性，一般不使用。

#### 2. `constructor`
指向 Number 构造函数本身。

### 三、常见实例方法
#### 1. toFixed(保留小数位数)
- 作用：保留指定小数位数，**四舍五入**，返回字符串类型(重点)
- 语法：
```javascript
数值.toFixed(小数位数)
```
```javascript
const price = 12.345
// 保留 2 位小数
console.log(price.toFixed(2)) // '12.35'
// 保留 0 位小数，取整
console.log(price.toFixed(0)) // '12'
```

#### 2. toString(进制)
- 作用：将数值转为字符串，可指定转换进制，默认十进制
- 语法：
```javascript
数值.toString(进制)
```
```javascript
const num = 10
console.log(num.toString())  // '10' 默认十进制
console.log(num.toString(2)) // '1010' 转为二进制
console.log(num.toString(8)) // '12' 转为八进制
console.log(num.toString(16))// 'a' 转为十六进制
```

#### 3. toPrecision(有效数字位数)
- 作用：按指定有效数字位数格式化数值，返回字符串
- 语法：
```javascript
数值.toPrecision(有效数字位数)
```
```javascript
const n = 123.45
console.log(n.toPrecision(3)) // '123'
console.log(n.toPrecision(4)) // '123.5'
```

### 四、常见静态方法（直接通过 Number 调用）
#### 1. Number.isInteger()
- 作用：判断一个值是否为**整数**，返回布尔值
```javascript
console.log(Number.isInteger(10))    // true
console.log(Number.isInteger(10.5))  // false
```

#### 2. Number.isNaN()
- 作用：判断一个值是否为 NaN，仅对真正的 NaN 生效
```javascript
console.log(Number.isNaN(NaN))  // true
console.log(Number.isNaN(123))  // false
```

#### 3. Number.parseInt()
- 作用：从数值/字符串中**提取整数**，从左向右解析，遇到非数字停止
```javascript
console.log(Number.parseInt('123abc')) // 123
console.log(Number.parseInt(3.99))     // 3
```

#### 4. Number.parseFloat()
- 作用：从数值/字符串中**提取浮点数**，支持解析小数
```javascript
console.log(Number.parseFloat('3.14px')) // 3.14
console.log(Number.parseFloat('abc66'))  // NaN
```
