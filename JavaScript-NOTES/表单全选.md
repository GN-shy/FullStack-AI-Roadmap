# 表单全选

## 一、完整可运行代码
```html
<!DOCTYPE html>
<html lang="zh-CN">
<body>
  <!-- HTML结构 -->
  <input type="checkbox" id="checkAll">全选
  <br>
  <input type="checkbox" class="ck">选项1
  <input type="checkbox" class="ck">选项2
  <input type="checkbox" class="ck">选项3

  <script>
    // 1. 获取大复选框（全选框）
    const checkAll = document.querySelector('#checkAll')
    // 2. 获取所有小复选框
    const cks = document.querySelectorAll('.ck')

    // 3. 全选框控制所有小复选框
    checkAll.addEventListener('click', function () {
      // 遍历所有小复选框，同步选中状态
      for (let i = 0; i < cks.length; i++) {
        cks[i].checked = this.checked
      }
    })

    // 4. 小复选框控制全选框
    for (let i = 0; i < cks.length; i++) {
      cks[i].addEventListener('click', function () {
        // 已选中的小框数量 === 总小框数量 → 全选框勾选
        checkAll.checked = document.querySelectorAll('.ck:checked').length === cks.length
      })
    }
  </script>
</body>
</html>
```

---

## 二、逐行详细讲解
### 第一部分：获取元素
```js
const checkAll = document.querySelector('#checkAll')
```
- `document.querySelector('#checkAll')`：通过 **id选择器**，获取**唯一**的全选复选框
- `checkAll`：存储全选框元素

```js
const cks = document.querySelectorAll('.ck')
```
- `document.querySelectorAll('.ck')`：通过 **类选择器**，获取**所有**小复选框，返回类数组
- `cks`：存储所有小复选框集合

---

### 第二部分：全选框 → 控制小复选框
```js
checkAll.addEventListener('click', function () {
  for (let i = 0; i < cks.length; i++) {
    cks[i].checked = this.checked
  }
})
```
1. `checkAll.addEventListener('click', function(){})`：给全选框绑定**点击事件**，点击后执行函数
2. `this.checked`：在事件函数里，`this` 指向**点击的全选框**，`checked` 是复选框属性，值为 `true(选中)/false(未选中)`
3. `for (let i = 0; i < cks.length; i++)`：循环遍历每一个小复选框
4. `cks[i].checked = this.checked`：
   - 左边：第 `i` 个小复选框的选中状态
   - 右边：全选框当前的选中状态
   - 作用：**小复选框和全选框状态完全同步**

---

### 第三部分：小复选框 → 控制全选框（双向联动）
```js
for (let i = 0; i < cks.length; i++) {
  cks[i].addEventListener('click', function () {
    checkAll.checked = document.querySelectorAll('.ck:checked').length === cks.length
  })
}
```
1. 外层循环：给**每一个小复选框**都绑定点击事件
2. `document.querySelectorAll('.ck:checked')`：CSS选择器，**只筛选被勾选的小复选框**
3. `.length`：获取**已勾选的小复选框个数**
4. `cks.length`：获取**小复选框总个数**
5. `===`：判断是否完全相等，返回布尔值
   - 全部勾选 → 等式成立 → `true` → 全选框自动勾选
   - 有1个没勾选 → 等式不成立 → `false` → 全选框自动取消
6. 关键：逻辑必须写在**点击事件内部**，每次点击都实时获取最新勾选状态

---

## 三、核心逻辑总结
1. 点击**全选框**：所有小框同步选中/取消
2. 点击**任意小框**：自动判断是否全部勾选，控制全选框状态
3. 实现**双向联动**，和网页购物车全选功能逻辑完全一致
